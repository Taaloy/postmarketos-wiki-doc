Model CPU GPU Notes
MSM7225/MSM7625 1x528MHz ARM11 (ARMv6) Software rendered The earliest S1 chips; missing a hardware floating point, thus unable to run postmarketOS (see Unsupported Devices).
MSM7227/MSM7627 1x800MHz ARM11 (ARMv6) Adreno 200 Later ARMv6 S1 chips; unlike their predecessors, these do have a hardware floating point.
