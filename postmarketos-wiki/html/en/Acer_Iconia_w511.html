Disable secure boot

update your uefi firmware to V2.0 and your 3G firmware. Else you may later need to go back to windows to do so

1. Press Vol+ and Vol- and Power buttons at the same time to boot into the UEFI. 2. Set a supervisor password and make sure to note it down. 3. Now disable secure boot in the settings

Booting

Using the genereci tablet x86 port does not boot on this device. Due to some forums it may need a fat32 boot partition and no fat16 one.
