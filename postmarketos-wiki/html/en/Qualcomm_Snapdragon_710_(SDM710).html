The Snapdragon 710 is a mid-range Qualcomm Soc released in 2018.

Mainline

Currently mainline refuses to boot (testing on Realme 3 Pro).

Notes

  • It appears to be very similar to the sdm670
  • It is built on a better manufacturing node (TSMC vs Samsung), which accounts for slightly bumped clocks.
  • It has a bumped GPU name adreno 616, however they appear very similar.
  • There is a modem update from x12 to x15.
  • Apart from this they share mostly the same things, and a diff on the outputted dtc files shows above differences plus some audio and pin changes.
  • All downstream sdm710 kernels inherit from sdm670.dtsi and friends.
  • It might be better to base a mainline attempt off sdm670 based kernel / inherit from upstream sdm670.dtsi?
