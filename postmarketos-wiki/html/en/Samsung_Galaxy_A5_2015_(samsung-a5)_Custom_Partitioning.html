Erasing unused partitions/Custom formatting

WARNING: After performing these steps, it will be IMPOSSIBLE to run ANY downstream OS, including, but not limited to One UI stock ROM , Samsung Experience stock ROM , TouchWiz stock ROMS, ANY custom Android ROMs, ANY Halium-based Linux distro, or anything else not using a MAINLINE Linux kernel.

It is safe to wipe partitions /dev/mmcblk0p18-24,/dev/mmcblk0p26 without any harm being done to the bootloader. Erasing these partitions will leave only mainline kernels bootable.

If you dont need recovery mode you also could remove /dev/mmcblk0p25 and /dev/mmcblk0p17.

If you dont need android boot partition you also could remove /dev/mmcblk0p16.

Reverting to stock from this setup

To revert to stock, Download the stock rom and flash it via heimdall.
