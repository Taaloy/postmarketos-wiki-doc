Installation

1. Use: pmbootstrap install --android-recovery-zip --recovery-install-partition=data
2. Copy pmos-samsung-a30.zip to external sdcard
3. Boot into TWRP
4. Format Data
5. Wipe Dalvik, Cache, Data
6. Mount External SD Card Only
7. Install zip file (If you see failed to mount /data, ignore it)
8. Reboot to system
