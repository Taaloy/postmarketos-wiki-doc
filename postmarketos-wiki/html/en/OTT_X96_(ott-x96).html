      See the SoC page for common tips, guides and troubleshooting steps

      Installation

      Use pmbootstrap to build your own image, like so 

      $ pmbootstrap init # select manufacturer: librecomputer, codename: lepotato

      Then make a local installation with required packages 

      $ pmbootstrap install --add u-boot-amlogic-s905x-tv,soc-amlogic-s905-boot

      Enter pmbootstrap chroot 

      $ pmbootstrap chroot -r

      Set deviceinfo variables in chroot 

      # echo '''
deviceinfo_dtb="amlogic/meson-gxl-s905x-p212"
deviceinfo_disable_dhcp="true"
''' >> /etc/deviceinfo

      Make sure variables are in place 

      # cat /etc/deviceinfo

      Exit pmbootstrap chroot, this should return to your regular shell 

      # exit

      Install to SD card: 

      $ pmbootstrap install --disk /dev/sdX # replace X with correct letter

      Connect SD Card and power on device holding reset button which is in AV port

      WiFi

      Needs proper device tree and this driver https://github.com/rehsack/rtl8189ES_linux
