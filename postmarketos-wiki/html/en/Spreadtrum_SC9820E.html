Unisoc (Spreadtrum) SC9820E is a system-on-chip for low-end devices, such as

  • Android-powered feature phones
  • Smart watches
  • KaiOS and KaiOS Touch phones

Current status in pmOS

Boots with armv7 downstream generic kernel (4.4.x version). Display output is not working out-of-the-box at this moment. Mainline is not tested.

Mainline potential

Given that there is basic mainline support for aarch64 Spreadtrum and GPU is supported by Panfrost driver, it is theoretically possible to run mainline kernel with hw acceleration support on devices with this SoC.
