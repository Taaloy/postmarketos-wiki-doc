      The Broadcom BCM2711 is the SoC used in the Raspberry Pi 4, Raspberry Pi 400 and the Raspberry Pi Compute Module 4 (CM4). The Pi is fairly well supported by Alpine Linux, which postmarketOS is built upon. See the Raspberry Pi 4 wiki page for more information.
