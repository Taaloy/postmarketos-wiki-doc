Animatch

From postmarketOS
Jump to navigation Jump to search

Animatch is a cute match-three game designed to work on Linux phones, originally for Purism's Librem 5 .

Installation

Animatch is avaiable in Alpine's community repository and can be installed via the following command (at the time of writing, only on Edge): 

# apk add animatch
