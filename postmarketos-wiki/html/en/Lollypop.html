Lollypop

From postmarketOS
Jump to navigation Jump to search

Lollypop is a mobile-friendly music player made in GTK 3. Worth noting is that it inhibits suspend, so that your phone keeps playing instead of suspending while it's playing music.

Installation

Lollypop is available in Alpine's testing repository and can as such be installed via the following: 

# apk add lollypop

Media format support

Instructions on the page for vvave apply to Lollypop as well.
