KTrip

From postmarketOS
Jump to navigation Jump to search

KTrip is a KDE application for querying public transport journeys between locations. It was made with mobile Linux in mind and fits well on a phone screen.

Installation

KTrip is available from postmarketOS's repository and can as such be installed directly via apk: 

# apk add ktrip
