Driver

Offical FL2000 Driver

This is a out of tree kernel module for those gpu . That driver isnt a drm driver so you cant use this driver for video out but you can use it for radio transmitting.

FL2000 DRM Driver

This is a out of tree kernel module for those gpu . That driver is a drm driver so you can use this driver for video out
