How to enter flash mode

  • Hold Volume Down, Home and Power buttons
  • When the warning screen comes up, press Volume Up to continue

Installation

  • 1. Flash the kernel pmbootstrap flasher flash_kernel
  • 2. Flash system image pmbootstrap flasher flash_rootfs
  • 3. Install system image to sdcard pmbootstrap install --sdcard /dev/mmcblkXXX
