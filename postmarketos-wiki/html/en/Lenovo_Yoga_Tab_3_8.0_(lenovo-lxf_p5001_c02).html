How to enter flash mode

  • Power on the device with Power and Volume down

Installation

  • Defconfig can be extracted from the stock boot.img
  • CONFIG_RADIO_IRIS_TRANSPORT needs to be disabled to build the kernel
  • A patch is required to build the kernel (fixes linux/types.h: no such file or directory in byteshift.h files)
