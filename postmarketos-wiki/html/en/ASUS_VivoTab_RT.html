Installation

Booting via fusee gelee and using re-crypt for permanent u-boot installation are both successful.

Hardware status (mainline)

Status Hardware Info
Y Booting Booting is available via loading custom U-Boot and kernel via fusee gelee.
Y U-Boot U-Boot is fully functional, can be loaded and flashed instead of UEFI.
N Panel Hydis HV101HD1 LCD SuperIPS+ HD panel through DSI-A port. Driver is made, DSI in T30 is not supported.
Y HDMI output HDMI works, but output is 1080p due to Tegra 3 limitations.
Y Touchscreen Used Atmel MXT1664 touchscreen.
Y USB One full size USB 2.0 in dock or via 36 pin adapter.
Y SD cards MicroSD card and full sized SD card (recognized as USB drive) in dock.
Y Internal memory eMMC by SanDisk. SPI flash W25Q32 4MB.
N GPS Broadcom GPS BCM4751 is used. BCM4751 GPS is commonly used in devices of this age, although it still does not have proper implementation (nor kernel, nor userspace);
Y WiFi Uses Azurewave AW-NH665 BCM4330.
Y Bluetooth Uses Azurewave AW-NH665 BCM4330B1.
Y GPIO keys There are 3 keys on GPIO: Power, Volume up, Volume down.

Additionally GPIO-keys like are LID sensor and Audio dock line-out detect.

Y Vibro VivoTab RT has GPIO based vibrator.
Y Dock Keyboard and Touchpad Dock keyboard and touchpad are both hid-over-i2c devices on hot-plug i2c line. Both are configured and work.
Y Battery/Charger Driver is based on Texas Instruments bq24725 SMBus Charge Controller driver but works through pad/dock Nuvoton NPCE795LA0BX embedded controller. Charging and charge level of both batteries work as expected.
Y PMIC Transformers use Texas Instruments TPS659110 PMIC and TPS62361 core regulator.
P Sound Sound codec is RT5640. Kernel driver and tegra_rt5640 driver are in grate master.
Y Thermal sensor NCT1008 ON Semiconductor LM90 series thermometer.
Y Orientation sensors Include Invensense mpu6050 IMU and Asahi-Kasei AK8975 3-axis digital compass.
P Light sensor Driver for Capella CM3218 is present but initiation is failed.
P Torch MFD Tablet has separate MFD for rear camera flash - TI TPS61050/61052 Boost Converter, and it works sometimes.
P NFC NXP-44701 is bound, but not tested.
N Front camera
N Rear camera
P ULP GeForce GPU Acceleration partially works.
