Moment

From postmarketOS
Jump to navigation Jump to search

Moment is a glossy Qt Matrix client with reasonable convergence support, forked from Mirage . End-to-end encryption is supported.

Installation

Moment is available in Alpine's community repository and can as such be installed on postmarketOS Edge via the following: 

# apk add moment

On postmarketOS v21.12 and earlier, you can install Mirage instead as Moment is unavilable: 

# apk add mirage
