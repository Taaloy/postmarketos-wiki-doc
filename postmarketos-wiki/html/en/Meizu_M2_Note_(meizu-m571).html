How to enter flash mode

The device has to be with unlocked bootloader. https://www.youtube.com/watch?v=X2xwfxQ3ll4 Needs special software (SP Flash tool), drivers and phone to be turned off. Connecting phone to PC while software is running will allow software to "hook" it (when booting phone temporarily exposes itself) and flash custom firmware

Installation

Fastboot doesn't allow flashing on this phone.

pmOS can be installed either using a Recovery ZIP through TWRP or using MediaTek's SP Flash Tool.

Install using Recovery ZIP

Follow the Installation from recovery mode guide.
