How to enter flash mode

Hold down the Home button + Power Button + Volume Down Button to get into Download mode. Heimdall does not work; so install through adb sideloading

Installation

Follow the installation guide, and run: pmbootstrap install --android-recovery-zip and pmbootstrap export. Then run adb sideload /tmp/postmarketOS-export/pmos-samsung-j7eltetmo.zip while your phone is in sideload mode.

Like the Samsung Galaxy J7 (2016) msm-fb-refresher is a required package to run user interfaces other than weston
