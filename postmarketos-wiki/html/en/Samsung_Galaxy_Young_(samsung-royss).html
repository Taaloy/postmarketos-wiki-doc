How to enter flash mode

Press and hold Home + Volume Down + Power until confirmation dialog appear, then press Volume Up to confirm.

Installation

See Installation guide .

What works

  • Booting
  • USB Networking
  • Flashing
  • Touchscreen
  • SD Card installation
  • Proximity sensor

What does not work

  • Touchscreen on wayland, kwin, i3wm (see issues below)

Not tested

  • WiFi (need kernel modules) WiFi works but see issues below
  • Bluetooth
  • Accelerometer

Issues

  • Audio output only on earpiece (need config?)
  • Starting PulseAudio causes a reboot
  • WiFi works if the kernel modules compiled with gcc4 and needs firmware (I haven't package it yet)
