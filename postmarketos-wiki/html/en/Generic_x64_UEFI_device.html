Installation

We currently don't offer prebuilt images for generic x86/64 devices, so you will need to follow the Installation Guide to build an image from scratch. The process is usually fairly straight forward, select tablet as the manufacturer when prompted.

When instructed to run pmbootstrap install, insert your target medium (sdcard, USB thumbdrive, etc) and specify --sdcard /dev/sdX where /dev/sdX is the target device.

The on-device installer is not currently supported, so you will need to either install directly to your target storage device, or remount the rootfs as readonly after first boot and use a tool like dd to copy the disk image over to the internal storage device.
