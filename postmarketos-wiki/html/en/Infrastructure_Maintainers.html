GitHub Mirror

  • Synchronizes all repositories on the GitLab postmarketOS group to https://github.com/postmarketOS-mirror .
  • Scripts: lab2hub
  • Maintainer: minecrell
  • Has access: minecrell, ollieparanoid
  • Admin part (create new repositories, update repository settings) needs to be run manually, see lab2hub README.
  • Sync part runs once per hour on minecrell's server, backups of all Git repositories exist for a while back.

pmsourcedump (GitHub)

  • Various mirrors for open source code where the upstream link isn't useful for us (dead/not stable/one big tarball with too much stuff).
  • https://github.com/pmsourcedump
  • Maintainers: PureTryOut, MartijnBraam, ollieparanoid
  • Has access: Danct12 to https://github.com/pmsourcedump/firmware-fxtec-qx1000
