How to enter flash mode

  • Fastboot: Power on the device with both Power and Volume Up buttons held down. Then select Fastboot by pressing Volume Up and confirm with Volume Down.
  • Recovery: Do the same as for Fastboot, but select the Recovery option in the menu instead.
  • MediaTek Preloader: The preloader triggers automatically everytime the phone turns on or reboots.

Installation

fastboot booting is broken on this device, so kernels will always have to be flashed to the boot partition of the device.

LEDs

Currently you can control two LEDs present on the device via sysfs by writing values to the brightness control files:

Screen brightness (0-255) can be adjusted: echo 255 | sudo tee /sys/class/leds/lcd-backlight/brightness

Rear camera flashlight state (0 or 1) can be toggled: echo 1 > /sys/class/flashlight_core/flashlight/flashlight_torch
