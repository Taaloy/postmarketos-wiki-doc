How to Install

Note: Unfortunately this device was removed from postmarketOS. See "Device removed" under see also for more information.

The Model has built-in fastboot mode, but can't use for installing pmOS, need lk2nd to boot the generic boot.img... For Other model which is not equipped fastboot, need to install older version of "stock aboot image" by QFIL first.

  1. Install lk2nd (alternative bootloader that provides a standard fastboot interface)
  2. Follow Qualcomm_Snapdragon_410/412_(MSM8916)#Installation .

Bluetooth seems work on phosh
