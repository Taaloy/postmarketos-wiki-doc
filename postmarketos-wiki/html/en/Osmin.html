osmin is a offline map application similar to osmand.

Installation

osmin is shipped with Alpine (osmin), thus it can be easily installed in pmOS: 

# apk add osmin

Mobile version

Using osmin

Befor being able to do anything in osmin you have to download a map of the area you want to view. To be able to get your location geoclue has to run you may have to start it manuelly
