Display

While simple uncompressed panel drivers, such as on Xiaomi Pad 5, work, display stream compression (DSC) is not working. This is likely because the hardware on this chipset is different from the implementation on SDM845, which means that a driver for this will need to be ported over from the android kernel. This leaves Xperia 5 II (sony-pdx206) with only framebuffer display for the moment.
