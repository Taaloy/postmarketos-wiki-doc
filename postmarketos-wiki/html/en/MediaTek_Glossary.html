A

  • AFE - Analog front-end
  • APC - see devapc below
  • APDMA - Application Processor Direct Memory Access
  • APMIXEDSYS - Application Processor Mixed System (?)
  • AUXADC - Auxillary Analog-to-Digital Converter

B

  • BTIF - Bluetooth Interface (serial interface similar to UART)
  • BROM - Boot ROM (read-only memory)

C

  • CCI - Cache Coherent Interconnect
  • CCIF - CPU-CPU Interface
  • CQDMA - Command Queue Direct Memory Access

D

  • DCM - Digital Clock Manager
  • DCQ - Display Command Queue
  • devapc - Device Automatic Power Control
  • DispRDMA - Display Read Direct Memory Access
  • DISPSYS - Display System
  • DVFS - Dynamic Voltage Frequency Scaler

E

  • EINTC - External Interrupt Controller
  • EMI - External Memory Interface

F

  • FHCTL - Frequency Hoping Control

G

  • GPT - General-purpose timer

H

I

  • INFRACFG - Infrastructure System Configuration
  • INFRASYS - Infrastructure System

J

K

L

  • LARB - Local Arbiter

M

  • MCI - Mediatek Coherence Interface
  • MCUSYS - Microcontroller Unit System
  • MDSYS - Modem System
  • MFG - MFlexGraphics (GPU system)
  • MMSYS - Multimedia System
  • M4U/MMMMU - Multimedia Memory Management Unit
  • MNL - Mediatek Navigation Library
  • MTKFB - MediaTek Framebuffer
  • MSDC - Memory Stick and SD card Controller

N

O

  • OVL - Overlay Engine

P

  • PERICFG - Peripheral System Configuration
  • PERISYS - Peripheral System
  • PWRAP - PMIC (Power management integrated circuit) Wrapper

Q

R

  • RSZ - PQ Engine Resize

S

  • SMI - Smart Memory Interface
  • SVS - Smart Voltage Scaling
  • SCPSYS: System Control Processor System
  • SYS_CIRQ - System Interrupt Controller

T

  • TOPCKGEN - Top Clock Generator
  • TOPRGU - Top Reset Generation Unit

U

V

W

  • WDT - Watchdog

Y
