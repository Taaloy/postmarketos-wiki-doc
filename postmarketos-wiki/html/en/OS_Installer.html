OS Installer

Install or try postmarketOS on many laptops and desktop computers.

You need an empty USB drive/SD card and a computer to flash the OS Installer to it.

Usage

Download

Download the latest image.

You must choose an OS Installer image that is compatible with your device. The link above is for generic x86_64 systems, but other systems may have their own installer images. See the device wiki page for your device for more information.

Install Flasher

  • Linux: Gnome Disks (Drive Options > Restore Disk Image)
  • Windows: Rufus
  • Mac: Balena Etcher

Flash to USB

Flash the postmarketOS-edge-os-installer-8-generic-x86_64-lts.img.xz with the flasher tool to the USB drive.

Reboot

Reboot your device to USB, you might enter the Bios to do so:

  • x86: hold Delete or F4 key
  • MacÂ : hold Alt key

and select boot into USB drive.

Install

It boots into a GNOME-based live system where you can try postmarketOS on your device, while changes stay persistent on the USB drive, your computer stays untouched until you follow the steps of the installer.

You need internet for this procedure!
