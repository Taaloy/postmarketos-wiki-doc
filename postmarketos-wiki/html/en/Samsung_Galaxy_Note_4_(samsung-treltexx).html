Please note that this page is about the models with Exynos chipset SM-N910C/H/K/L/S/U. The Note 4 variants with Snapdragon 805 has codename trltexx .

There are currently no port for this device, or the sibling samsung-tre3calte . A tablet based on exynos5433 ( samsung-gts210lte ) is supported though. It should be possible to just copy gts210lte's kernel package , and modify the config, to get a working kernel for treltexx.
