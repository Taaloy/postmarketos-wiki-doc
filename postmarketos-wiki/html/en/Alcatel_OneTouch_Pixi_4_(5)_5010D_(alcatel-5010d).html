How to enter flash mode

TODO Explain TWRP recovery installaton process

To enter recovery, hold Power (until logo appears) and Volume up buttons.

If you want to enter fastboot (can't flash system), go Reboot → Bootloader .

Installation

  1. Install pmbootstrap.
  2. pmbootstrap init
    • Vendor must be alcatel
    • Device codename must be 5010d
  3. pmbootstrap install --android-recovery-zip
  4. pmbootstrap flasher --method adb fastboot
  5. Go to recovery (described above) → Advanced → ADB Sideload → Swipe to Start Sideload
  6. Connect device to your PC
