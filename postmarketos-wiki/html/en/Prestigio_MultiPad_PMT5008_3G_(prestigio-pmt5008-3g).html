Installation

Mainline Kernel Status

There is no downstream kernel source code for this device. So I decided to port mainline linux and I was pretty quick to successfully start 4 uart ports, wdt, smp and /bin/sh from initramfs. But for further progress, you need to write the pinctrl and clk drivers.

Also this device uses pmic mt6323 which seems to be in mainline.
