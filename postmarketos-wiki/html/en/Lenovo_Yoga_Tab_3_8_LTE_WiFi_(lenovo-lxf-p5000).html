The Lenovo Yoga Tab 3 8 is 8 in version of Lenovo Yoga Tab 3 10 LTE/WiFi (lenovo-lxf-p5100).There are QR code sticker that has a code name printed on it.The sticker is sticked onto the motherboard.All Lenovo Yoga Tab 3 8 varian code name appear to have lxf-p5000 prefix and a lot of different suffix. Known suffixes :

_C13 : Model Name yt3-850f,WIFI Variant

_C12 : Model Name yt3-850f,WIFI Variant
