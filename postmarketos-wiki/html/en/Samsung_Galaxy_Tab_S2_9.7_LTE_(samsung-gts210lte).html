What works

  • Touchscreen
  • USB network

Unlike on many other Samsung devices, wifi does not work out of the box after adding firmware files. The firmware package that exists for the device provides all the wifi firmware files that should be necessary, but do not actually make wifi work (as of now).

Installation

Installation can be done with the standard installation instructions. Heimdall cannot flash the rootfs though, so if you want to install to the internal storage you need to do that with the --android-recovery-zip method (generate a zip and install it through twrp/recovery).
