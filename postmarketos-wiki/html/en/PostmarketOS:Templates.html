From postmarketOS
Jump to navigation Jump to search

MediaWiki allows us to define custom templates, which can then be included in other wiki pages. When creating a new one, please add it below, and use the " Template:Template " at the top of the new template. Check existing templates for reference. An automatically generated list of templates can be found here .

Available templates

  • alpine link to a page on the Alpine Wiki
  • albug link to a bug over at https://bugs.alpinelinux.org/
  • button Visual representation for a button or sequence of buttons
  • app format several standard properties of an app, used in Applications by category
  • appantifeature / appaf describe properties of an app that do not adhere to postmarketOS' principles, used in Applications by category
  • file gitlab or github file link
  • github github issue or pull-request link (obsolete for postmarketOS code, use issue or MR instead)
  • hc block code with header (can be used for listing the contents of a file, with the file location on top)
  • icon box simple box with icon
  • side box smaller simple box without icon
  • infobox on the top right of every device / user interface article
    • interface
    • device
    • soc
  • based-on-soc sidebox that directs to SoC page for common information
  • issue gitlab issue link
  • MR gitlab merge request link
  • maemo-leste Maemo Leste bugtracker link
  • mainpage-button fancy buttons with icons used on the Main Page
  • milestone GitLab milestone link
  • note
  • note-device-milestone-main WIP for main device category with milestone link
  • outdated
  • syslog short explanation on how to read relevant syslog entries
  • todo
  • userbox userbox template
  • warning
  • wikipedia link to an wikipedia article

Templates for device ownership list:

  • My devices Show table of devices owned on your own page
  • Owns device Adds a device to the ownership table
  • Device owners Displays list of people owning the device to the device page

Informative boxes for pages:

  • Template:Stub

Userboxes:

  • Template:User_ports Shows the amount of devices you ported (usage: {

    {User ports|amount}})
  • Template:User_main_device Shows the user's main device (usage: {

    {User main device|Wiki link}})
  • Template:User_mainline Shows the amount of devices you mainlined (usage: {

    {User mainline|amount}})
  • Template:User_mainline_attempts "This user is currently trying to mainline a device."
  • Template:User_Matrix Links to user's Matrix profile (usage: {

    {User Matrix|@username:homeserver.org}})
  • Template:User_GitLab Links to user's GitLab profile (usage: {

    {User GitLab|username}})
  • Template:User_GitHub Links to user's GitHub profile (usage: {

    {User GitHub|username}})
