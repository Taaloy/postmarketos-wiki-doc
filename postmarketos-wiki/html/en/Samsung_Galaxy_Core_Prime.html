Hardware revisions

The Galaxy Core Prime is not a single type of device. The different versions should probably have separate pages in the future, but for now it's a single table. GSMArena lists quite a few different typenumbers, but the exact difference is unclear. This table is based on a PDF found here . Correctness is not guaranteed because it is based on online information. User:Eloy has verified the existence of a Marvell PXA1908 version, the coreprimevelte.
