WARNING: Currently,Postmarket OS doesn't officially support them.

MS9132/MS912x (MS912x is for usb2/MS9132 is for usb 3) are GPU chips that allow to connect monitor via usb.

MS912x Driver

This is a out of tree kernel module for those gpu . That driver is a drm driver so you can use this driver for video out.
