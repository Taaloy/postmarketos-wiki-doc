How to enter flash mode

  • Download mode: While the device is off hold Power, Volume Down and Home button to boot it into download mode, afterwards confirm the operation by pressing Volume Up.
  • Recovery mode: While the device is off hold Power, Volume Up and Home button to boot it into recovery mode.

Installation

Work in progress device and kernel packages can be found here.

Currently, the device gets stuck in a bootloop and only boots sometimes.

Mainline

According to Documentation/arm/marvell.rst the PXA1908 chipset should be supported, but it is unknown whether it actually works.

Booting

The SM-G531F uses the Marvell PXA1908 chipset, which utilizes an uncommon way to specify the revision in DTB files, therefore the dtbTool from https://github.com/osm0sis/pxa-mkbootimg should be used. This is the same with mkbootimg. Additionally, SEANDROIDENFORCE should be appended to the bootimage, in order to silence the "KERNEL IS NOT SEANDROID ENFORCING" warning.
