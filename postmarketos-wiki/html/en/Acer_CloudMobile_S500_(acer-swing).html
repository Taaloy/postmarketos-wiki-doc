Installation

Initial, you need to unlock the bootloader. Now, it's preferable to install pmOS on SDCard and boot the kernel without purging stock Android. 

pmbootstrap install --sdcard=/dev/sdX
pmbootstrap flasher boot

Turn on the backlight: 

ssh user@172.16.42.1
echo 91 | sudo tee /sys/class/leds/lcd-backlight/brightness
