How to enter flash mode

To flash with heimdall hold power + volumedown + home

To flash with recovery hold power + volumeup + home

Installation

Follow Installation Guide Run pmbootstrap install --android-recovery-zip --recovery-install-partition=data Run pmbootstrap export Copy the recovery zip file to a sd card and insert in into the phone to flash with twrp
