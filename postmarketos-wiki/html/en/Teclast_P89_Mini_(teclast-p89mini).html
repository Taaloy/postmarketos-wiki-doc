Bootloader

This device (just like many other SFI devices) uses Droidboot, which is a fastboot implementation. The bootloader is permamently locked, however there is a way to overcome this by signing a boot image (see: https://github.com/alef78/intel-ifwi-study )

Mainline kernel

There's a chance this device could run a kernel 5.11 (or older). Newer kernels dropped SFI and cannot be ran without porting the SFI platform support over there.
