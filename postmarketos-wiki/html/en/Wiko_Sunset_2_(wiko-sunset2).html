How to enter flash mode

  • Start SP Flash Tool
  • Select the MT6572 scatter file
  • Check the partition to flash
  • Specify the location of the file to flash
  • Click Download button
  • Remove the battery from the device
  • Connect the USB cable
  • Flash process should start

How to enter Recovery Mode

  • Hold the Volume Up and Volume Down button, and then press the Power button
  • Press Volume Up when prompted

How to enter Factory Mode

  • Hold the Volume Up and Volume Down button, and then press the Power button
  • Press Volume Down when prompted
