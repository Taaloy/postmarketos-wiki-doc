How to enter recovery mode

  • Power + Vol+

Unlocking

There is no working Fusée Gelée version and u-boot port yet

Sony does not seem to have provided a way to enter APX using a key combination, so we have to use an alternative method. Use the following project for rooting and entering APX mode: https://gitlab.com/eloydegen/apxploit
