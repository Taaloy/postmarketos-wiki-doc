A light, highly tablet-friendly (not strictly mobile-friendly!) Linux desktop initially developed for the CutiePi Raspberry Pi tablet using Qt/QML.

Offers a QtWayland compositor (video) and other small UIs such as a custom terminal, web browser, lock screen, wireless/quick switches and on-screen keyboard (visible here).

The shell is currently hardcoded in a couple of places to use RPi-specific settings, see upstream issue #1 for more information.
