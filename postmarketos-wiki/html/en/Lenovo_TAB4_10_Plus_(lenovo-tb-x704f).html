How to enter flash mode

When shut down, hold volume up and power button until it starts

Installation

(after initializing pmbootstrap and choosing device, assumes device is already in fastboot) 

$ pmbootstrap build linux-lenovo-tb-x704f
$ pmbootstrap build device-lenovo-tb-x704f
$ pmbootstrap install
$ pmbootstrap flasher flash_rootfs
$ pmbootstrap flasher flash_kernel

There is a significant time gap between boot and SSH becoming available.
