Topwise A721 is motherboard that can be found in multiple devices, including LY-F1, Tracer OVO 1.2 or Gooseberry Single Board Computer. As the device uses Allwinner A10 SOC, it is fully capable of booting from SD Card. Changes allowing it to run Mainline Linux kernel are currently queued for 5.13.

Additional Info

Clock Work Mod

CWM for the device can be found on Shanduur's Google Drive To install it, firstly get Android Terminal Emulator (you will probably have to install it with adb install ) or perform this action directly from adb shell . Those devices are rooted by default. In the terminal, copy the files from SD-Card ( /mnt/extsd ) to internal storage ( /mnt/sdcard ). Then run following: 

$ su
# cd /mnt/sdcard
# sh install-recovery.sh
# sh reboot-recovery.sh
