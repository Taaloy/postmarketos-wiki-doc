This is a UEFI base second-stage bootloader targetting mobile devices powered by Qualcomm Snapdragon SoC and aiming to provide an usable EDK2 UEFI enviroment for these devices.

It can be used as a boot manager for multi-booting mainline Linux, Android and optionally Windows on certain SoCs.
