      From postmarketOS

      The PinePhone mostly works with the vanilla kernel. However, postmarketOS is using megi's kernel because some parts are not completely upstreamed yet.

      The following table lists all branches in megi's kernel and how they relate to the pinephone.

      Branches in megi's kernel
      Branch Related to PP Comment
      8723cs-6.9 N Could eventually be replaced by rtw88 (currently in wireless-next)
      a711-6.9 N A711 tablet
      a711-cams-6.9 N A711 tablet
      a711-chgled-6.9 N A711 tablet
      a711-vibrator-6.9 N A711 tablet
      a83t-suspend-6.9 N A711 tablet
      anx-6.9 Y
      ap6275p-6.9 N (?)
      arasan-6.9 N (?)
      audio-6.9 Y (?)
      aw-opp-6.9 N
      aw-suspend-6.9 N (?) Probably only required for wakeup on RTC when PP is powered down
      axp-6.9 Y
      bes2600-6.9 N Pinetap WiFi
      bt-6.9 Y Powersaving for bluetooth on sleep
      defcon-6.9 N Default configs
      drm-6.9 Y
      err-6.9 Y
      fixes-6.9 Y (?) Seems to contain minor things
      fusb302-6.9 N (?) There is one patch which corrects registers. Maybe we need that?
      goodix-6.9 Y Powersaving for touchscreen on sleep
      hdmi-audio-6.9 Y
      husb311-6.9 N Probably for some other device.
      imx258-6.9 N Probably for some other device.
      modem-6.9 Y (?) Powermanager for PP modem. Not sure, if pmOS uses this
      opi3-eth-6.9 N
      orange-pi-6.9 N Main integration branch that merges all other branches
      ov5640-6.9 Y (?) Not strictly needed, but probably increases stability
      pb-6.9 N Pinebook
      pbp-6.9 N Pinebook Pro
      pinetab-6.9 N Pinetab
      pp-6.9 Y Lots of small patches.
      ppkb-6.9 Y Support for pinephone keyboard
      ppp-6.9 N Pinephone Pro
      ppp-panel-6.9 N Pinephone Pro
      private-6.9 N Megi's personal stuff
      pt2-6.9 N Pinetab 2
      rk3399-typec-6.9 N Rockchip stuff, not used on OG PP
      rk3588-6.9 N Rockchip stuff, not used on OG PP
      rk817-6.9 N Rockchip stuff, not used on OG PP
      rk818-6.9 N Rockchip stuff, not used on OG PP
      rp64-6.9 N Rockchip stuff, not used on OG PP
      rt5640-6.9 (?) What is this?
      rtw88-6.9 Y Extends rtw88 for OG PP WiFi (rtl8723cs), this is already in wireless-next
      speed-6.9 N Technically yes, but just (unupstreamable?) boot time optimizations
      stk3310-6.9 N Not clear, if this is even used in the PP, but seems as if not strictly needed
      sun6i-csi-6.9 N (?) Not clear, if this is even used in the PP, but seems as if not strictly needed
      tcpm-6.9 N (?) Could be useful, but not strictly needed
      typec-extcon-6.9 Y I think we need this for external display
