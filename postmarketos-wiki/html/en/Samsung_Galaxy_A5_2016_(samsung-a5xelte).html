Users owning this device

How to enter flash mode

1. power off the device.

2. press Volume Down + Home + Power

3. at the blue screen press the Volume Up button to enter Odin download mode

Recovery

press Volume Up + Home + Power

Installation

Flash twrp heimdall flash --RECOVERY twrp-recovery.img to recovery. (Heimdall can make some problems, just retry multiple times with rebooting device)

Install recovery zip pmbootstrap install --android-recovery-zip .

Inside twrp, choose "ADB Sideload" and run pmbootstrap flasher --method=adb sideload
