Jailbreak

For firmware 3.65 - 3.74 h-encore² by Andy Nguyen must be used. For easier installation of h-encore² finalhe by soarqin can be used.

If you want a permanent jailbreak on 3.65 use the ensō installer

Downgrading firmware

Refer to modoru. This works on firmware 3.60 - 3.73.

Running Linux

Work is being done by xerpi on GitHub.
