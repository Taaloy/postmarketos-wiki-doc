How to enter flash mode

Plug USB micro to USB A cable into computer. With tablet turned OFF, press and hold vol. up and vol. down simultaneously, and plug USB micro end of cable into tablet. Release buttons when screen turns on.

Press vol. up to go to download mode.

Installation

Heimdall will not install kernel to boot partition. Upload completes but doesn't write to disk.

Can be installed using pmbootstrap install --android-recovery-zip and copying zip file to sdcard or usb-otg and installing from TWRP.

Installation completes successfully, but on rebooting goes to Download (odin) mode.

Notes

Kernel compiles with all pmbootstrap-supplied patches removed and patch https://github.com/Tomoms/android_kernel_oppo_msm8974/commit/11647f99b4de6bc460e106e876f72fc7af3e54a6 added.
