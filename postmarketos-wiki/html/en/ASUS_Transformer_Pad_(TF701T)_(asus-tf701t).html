Users owning this device

Preparation

Device needs to be unlocked before being able to flash using Unlock tool , at this time still available from Asus website

How to enter flash mode

boot with both "power" + "vol. down" buttons held down

Installation

Follow the instructions in [ installation guide ]. For now, please select xfce4 or mate as user interface in pmbootstrap init, as the other UIs fail at the moment

Flashing with "pmbootstrap flasher flash_rootfs --partition userdata" flashes the rootfs to the userdata partition leaving the most free space available.

Mainline

WIP branch: https://github.com/jenneron/linux-asus-tf701t

Status Hardware Info
Y Booting Boot partition is 8 MiBs so image size should be less than 8 MiBs.
Y Panel Sharp LQ101R1SX03 panel.
N HDMI output Bindings are present in device tree, but it is broken for now.
N Touchscreen Used Raydium RM31081 touchscreen connected via SPI. There is no driver in mainline.
P USB Gadget Dies after suspend
P USB Host Full size USB 3.0 port and SD slot in dock. Works only in EHCI mode.
Y SD card MicroSD in tablet.
N Internal memory HS200 eMMC
N GPS There is driver for BCM4751 which is similar to our BCM4752.
N WiFi Uses Azurewave BCM43341.
N Bluetooth Uses Azurewave BCM43341.
Y GPIO keys There is 3 keys on GPIO: Power, Volume up, Volume down and LID sensor
P Dock Keyboard, Touchpad and battery. For now only battery works. It is handled via Nuvoton NPCE69ALA0DX embedded controller.
Y Battery/Charger Charging and leds work as-is, battery status works with asuspec.
Y PMIC TPS65913B2B5
N Sound RT5639
N Thermal sensor NCT1008 ON Semiconductor LM90 series thermometer. Not tested
Y Inertial measurement unit Invensense mpu6500 accelerometer + gyroscope.
Y Magnetometer Asahi Kasei ak(m)09911. Mount matrix is not tested
Y Light sensor Dyna Image AL3320A.
N Front camera Aptina mi1040. VI is not supported yet.
N Rear camera OmniVision ov5693. VI is not supported yet. Driver is present, needs setup in device tree.
P ULP GeForce GPU Acceleration partially works.
