How to enter flash mode

  • Fastboot : Power on the device with both Power and Volume Down buttons.
  • APX : Power on the device with both Power and Volume Up buttons.

Installation

Follow Installation Guide .

Note that system partition is only 2 GB, so you probably want to do pmbootstrap flasher flash_rootfs --partition userdata .

Hardware Support

Grate Upstream Hardware Info
Y Y Booting Boot partition is 8 MiBs so image size should be less than 8 MiBs.
Y Y Panel Sharp LQ101R1SX03 panel.
N N HDMI output Bindings are present in device tree, but it is broken for now.
N N Touchscreen Used Raydium RM31081 touchscreen connected via SPI. There is no driver in mainline.
P P USB Gadget Dies after suspend
P P USB Host Full size USB 3.0 port and SD slot in dock. Works only in EHCI mode.
Y Y SD card MicroSD in tablet.
N N Internal memory HS200 eMMC
N N GPS There is driver for BCM4751 which is similar to our BCM4752.
N N WiFi Uses Azurewave BCM43341.
N N Bluetooth Uses Azurewave BCM43341.
Y Y GPIO keys There is 3 keys on GPIO: Power, Volume up, Volume down and LID sensor
P N Dock Keyboard, Touchpad and battery. For now only battery works. It is handled via Nuvoton NPCE69ALA0DX embedded controller.
Y N Battery/Charger Battery status works with asuspec.
Y Y PMIC TPS65913B2B5
N N Sound RT5639
N N Thermal sensor NCT1008 ON Semiconductor LM90 series thermometer. Not tested
Y Y Inertial measurement unit Invensense mpu6500 accelerometer + gyroscope.
Y Y Magnetometer Asahi Kasei ak(m)09911. Mount matrix is not tested
Y Y Light sensor Dyna Image AL3320A.
N N Front camera Aptina mi1040. VI is not supported yet.
N N Rear camera OmniVision ov5693. VI is not supported yet. Driver is present, needs setup in device tree.
P P ULP GeForce GPU Acceleration partially works.
