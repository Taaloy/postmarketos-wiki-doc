How to enter flash mode

Download mode: VolDown + power + home

Recovery mode: VolUp + power + home

Installation

android recovery zip: pmbootstrap install --android-recovery-zip

use pmbootstrap flasher --method=adb sideload or install manually with TWRP recovery (versions>3.0.0 fail to boot)

flashing with heimdall seems to fail

  • using Samsung's sparse image format flashes successfully, but /system partition won't mount (in TWRP at least) and needs to be wiped
