What Works

Nothing. (besides rooting and kernel building)

Using the lg-c800 device tree, I have begun work on building TWRP. It boots and adb is useable, but no further testing has been done.

What Does Not Work

Everything.

Rooting

The exploit used on the LG Esteem also works on the Doubpleplay. Please see this forum thread on AndroidCentral for instructions. [1]

The tool referenced has a dead download link, but is available through the Internet Archive's Wayback Machine. LG_Esteem_Root.zip on archive.org

The tool runs from a .bat script. Though the changes necessary to run it on linux are trivial, here is the modified script:
