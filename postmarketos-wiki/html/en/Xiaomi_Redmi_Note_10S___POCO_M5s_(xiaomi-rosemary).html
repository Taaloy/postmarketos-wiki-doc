How to enter flash mode

  • Fastboot: Power on the device with both Power and Volume Down buttons.
  • Recovery: Power on the device with both Power and Volume Up buttons.

Installation

WARNING: Make sure to backup any important data before you proceed
WARNING: Any deviation from the described installation process may result in bricked device
  1. Install pmbootstrap
  2. Run through pmbootstrap init and select 'xiaomi' as the vendor and 'rosemary-downstream' as the device
  3. Go through the rest of the setup with your preferences (don't use systemd)
  4. After the setup completes, generate a rootfs with pmbootstrap install
  5. Reboot into fastboot mode
  6. Flash kernel and rootfs with the following command: pmbootstrap flasher flash_kernel && pmbootstrap flasher flash_rootfs --partition userdata
  7. Reboot the phone: fastboot reboot

Unbricking

  • SP Flash Tool (XDA, English)
  • SP Flash Tool (4PDA, Russian)
  • MtkBypass (4PDA, Russian)
