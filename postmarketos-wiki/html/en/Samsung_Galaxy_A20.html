Installation

1. Use: pmbootstrap install --android-recovery-zip --recovery-install-partition=data
2. Copy to Internal SD-Card
2. Boot to TWRP
3. Wipe Dalvik, Cache, Data
4. Format Data -> YES
5. Unmount everything except external sd card.
6. Install the zip file from your sdcard. You will see errors "Failed to mount /data", don't worry about.
7. Now reboot to system
