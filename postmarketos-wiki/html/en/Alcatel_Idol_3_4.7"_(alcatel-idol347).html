How to enter flash mode

  • Recovery : Power on the device with both Power and Volume Up buttons.
  • Fastboot : Power on the device with both Power and Volume Down buttons.

Stock fastboot was removed on update from Android 5/Lollipop to 6/MarshMallow.
Installing lk2nd is required to run mainline kernel anyway.

Installation

  1. Install lk2nd (alternative bootloader that provides a standard fastboot interface) to boot partition using either:
    • Alcatel Flasher tool
    • emmcdl or edl.py with Idol34_7_NPRG7627.bin
    • dd command from TWRP
  2. Follow Qualcomm_Snapdragon_410/412_(MSM8916)#Installation .
