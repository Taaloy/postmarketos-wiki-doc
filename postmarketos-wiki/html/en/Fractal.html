Installation

Fractal is available in Alpine's community repository and can as such be installed as follows: 

# apk add fractal

Troubleshooting

Passwords are not saved

On some postmarketos installation, no keyring is created by default. This can be solved by downloading the seahorse package from alpine repos, pressing File > New > Password Keyring, and naming it login. You may have to use seahorse to unlock the keychain on each reboot, if your UI does not support unlocking keychains.

Crash by missing dependency gtksink

If Fractal crash when loading conversation and in the output to terminal is "Missing dependency element: gtksink ..." it can be solved by installing "gst-plugins-good-gtk".
