Installation

Notes

Device has been unlocked and is running TWRP.

To get the kernel to compile, all the default patches need to be removed and the following one added: https://github.com/Tomoms/android_kernel_oppo_msm8974/commit/11647f99b4de6bc460e106e876f72fc7af3e54a6

Need to add 

deviceinfo_flash_sparse="true"

to deviceinfo

On attempting to boot, the Lenovo logo appears but the tablet goes into fastboot mode and responds to fastboot commands.

The kernel source is older than any released system for this device, and is the only one available.Â :( This might partly account for failure to boot.
