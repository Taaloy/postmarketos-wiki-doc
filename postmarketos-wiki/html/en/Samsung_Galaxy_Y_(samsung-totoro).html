How to enter flash mode

Power+Vol+Up - recovery Power+Vol+Down - odin/download

Installation

Run pmbootstrap install (without installing to the SD card, since we need it to store the boot.img for flashing), run pmbootstrap export, flash the boot.img manually through TWRP, then reinstall to SD card.

Android ZIP install errors out, heimdall doesn't work due to a different flashing protocol.

Note: pmOS does not fit on the system partition. You have to use an SD card.

Parts list

Element type Element Works? Notes
Chipset BCM21553 Not supported in mainline (...yet :) )
Graphics Videocore IV Supported in upstream Mesa.
PMIC, USB switch, etc. MAX8986 Not in mainline, as far as I can tell
Display driver ILI9341 Supported in mainline as a drm panel.
Touchscreen Silabs F760, Cypress TMA140 TMA140 is supported, F760 is not. (On my device, the F760 appears to be used.)
WiFi, Bluetooth BCM4330 Driver in mainline
Backlight Skyworks AAT1401 Not in mainline
Camera SR200PC10 Not in mainline; in fact, I can't even find which company made the chip.
Proximity/ambient light sensor Sharp GP2A??? Not 100% sure which one; either way, should be supported in mainline
Accelerometer, magnetometer Bosch BMA222, MEMSIC MMC328X BMA222 is in mainline, MMC328X is not; there do seem to be other MEMSIC magnetometers in mainline, maybe they're similar?
