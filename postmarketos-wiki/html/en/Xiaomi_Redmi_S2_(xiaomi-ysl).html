Porting steps

  • start with lk2nd and mainline kernel
    you will spend a lot of time on downstream and not get any good results, cr aka_
  • Get into xiaomi repo and find board-ids for that device and clone lk2nd and take mido dts copy it rename and modify board-id, add to rules and try booting 
model = "Qualcomm Technologies, Inc. MSM8953 + PMI8940 MTP";
	compatible = "qcom,msm8953-mtp", "qcom,msm8953", "qcom,mtp";
	qcom,board-id= <8 0>;
	qcom,pmic-id = <0x010016 0x010011 0x0 0x0>;

might be problematic A9 bootloader

It probably won't boot but you can give it a try
