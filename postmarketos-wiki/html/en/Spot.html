Spot

From postmarketOS
Jump to navigation Jump to search

Spot is a Spotify client for GNOME that both supports phones and desktops. It uses GTK 4 and libadwaita, and can be installed either via apk from Alpine's community repository or via Flatpak from Flathub.

Installation

Via apk 

# apk add spot

Via Flatpak 

# flatpak install flathub dev.alextren.Spot
