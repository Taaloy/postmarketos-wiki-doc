Giara is a reddit app, built with mobile Linux in mind.

Giara is available from Alpine's community repository, and can as such be installed on postmarketOS v20.03 (and newer) and Edge via the following command: 

# apk add giara

Tweaks

Nothing happens when I try to log in

Ensure that you have a default browser set. You can check if you do by running the following: 

$ xdg-settings get default-web-browser
