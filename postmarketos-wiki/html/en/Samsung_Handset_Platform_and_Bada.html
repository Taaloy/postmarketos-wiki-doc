Flashing method

Samsung used a few different (Windows-only) flashing programs for their SHP/Bada phones. Some can be flashed with Samsung MultiLoader while others use programs like BcomDownloader.

Finding information

Service manuals usually contain a lot of valuable information about SHP/Bada phones. They may include block diagrams detailing the hardware and chipset used, schematics showing UART test points, and flashing instructions.

Existing projects

BadaDroid

BadaDroid was a project aiming to bring Android to the Samsung Wave (GT-S8500) and Wave 2 (GT-S8530). Android can be flashed on the internal storage, see here. Since this project uses a standard Android recovery, pmOS could probably be installed that way.

JétDroid

JétDroid was a project aiming to bring Android to the Samsung Jét (GT-S8000). Most resources about this project seem to be dead, but all the files are still available here. See this XDA thread for more information about JétDroid and its bootloader JetQi. The JetQi bootloader can boot a Linux kernel from a microSD card.
